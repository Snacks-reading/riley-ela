# Riley Reads & Spells — ELA (Phase 1)

## What this is
A clean, GitHub-Pages-friendly repo layout that:
- Locks **A1 Guard the Vowel (CVC Doubling)** as the baseline module (already working).
- Adds an **organized module registry** for the rest of Module A rules.
- Sets up placeholders for each module so the repo stays clean while we build one-by-one.

## Folder layout
- `index.html` — **Module launcher** (tiles like IXL/Prodigy style selection)
- `core/registry.js` — **single source of truth** for module list + routing
- `core/styles.css` — shared styling
- `assets/avatars/` — shared avatar assets
- `modules/` — each module has its own folder
  - `modules/A1_guard-the-vowel/` — LOCKED baseline build (v1_22_4)

## GitHub Pages
1. Put this repo on GitHub.
2. Settings → Pages → Deploy from branch → `main` (or `master`) → `/root`.
3. Visit the Pages URL.
