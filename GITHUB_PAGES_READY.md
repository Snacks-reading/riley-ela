GitHub Pages readiness

1. Create a public GitHub repository.
2. Upload the contents of this folder to the repository root.
3. In GitHub: Settings > Pages > Build and deployment > Deploy from a branch.
4. Choose Branch: main, Folder: /(root).
5. Wait for Pages to publish.
6. Your public URL will look like:
   https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/

After the site is live, generate a QR code that points to the public URL.
A working QR cannot be created until the public URL exists.
