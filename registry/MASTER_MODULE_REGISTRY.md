# Master Module Registry

This registry is the source of truth for Riley Reads & Spells V24.

## Tier 1
The Quest Board shows 19 categories as real book tiles. Each tile uses:
- a locked category id
- a locked title
- a locked book PNG
- a direct route to its Tier 2 page

## Tier 2
Star Speller uses real crest tiles rather than guessed image hotspots.

## Tier 3
Lessons run from one shared lesson engine with:
- one lesson state machine
- one speech queue
- one adaptive question queue
- one avatar state controller

## A2 Silent-E notes
Silent-e includes:
- long-vowel silent-e examples
- drop-e forms with -ing and -ed
- exceptions such as come, have, give, love, some, done
